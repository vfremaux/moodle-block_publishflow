Retrofitting courses to factory

Retrofitting will copy this course to the designated factory instance in the network

When launching the retrofitting operation, you will copy the structure and content of this course as per the last backup stored in the course files.

The backup used for retrofitting MUST IGNORE any user related information and data of the course, the activities, resources and blocks being reproduced with the same settings than the original.

The retrofitted unit will be available in a predefined category, unless the option of choosing the target category is open.
